# Description

**Project:**   Travel agency
To run the project you have to follow a few steps:

  

## Installation

 - Enter a project folder
 - Install all packages necessary for the project:

```bash

$ npm install

```
- Create new postgreSQL database: ***travel_agency_db***

## Database seeding

```bash

# services
$ npm run servicesSeed

```

## Running the app

  

```bash

# development
$ npm run start
  
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

  

## Test

  

```bash
# unit tests
$ npm run test
  
# e2e tests
$ npm run test:e2e
  
# test coverage
$ npm run test:cov

```
